Royal Perfumes is a sleek, responsive web application showcasing luxury perfumes. The website is designed with a royal theme using black, gold, and elegant gradients, providing a modern, mobile-friendly user experience. It includes a dynamic product catalog, WhatsApp ordering, and an admin panel with login functionality for managing products.

Features

Responsive Design: Optimized for mobile, tablet, and desktop screens.

Animated Admin Login: Click the "Admin Login" button to smoothly toggle the login panel.

Product Showcase: Display products with images, names, and prices.

Admin Panel:

Secure login (hardcoded for demo purposes)

Add new products with name, price, and image URL

Remove existing products

WhatsApp Integration: Quick access to order via WhatsApp.

Local Storage Support: Products persist in the browser even after page reload.

Interactive Animations: Smooth hover effects and sliding panels for modern UX.

Technologies Used

HTML5

CSS3 (with gradients, shadows, and animations)

JavaScript (DOM manipulation, localStorage, animations)
